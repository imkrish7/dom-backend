## Required Env variables

```env
  PERPLEXITY_URI=https://api.perplexity.ai/chat/completions
  PERPLEXITY_API_KEY=<perplexity api key>
```

## Description
A perplexity AI based project

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
